## <a name="custom-commands">Команды</a>:

`cmd(cmdAlias,...,cmdAlias) [arg(1) \~arg(2) ... arg(n)]` - функция команды

В [...] находятся аргументы команды, если аргумент помечен тильдой (\~), то он не обязателен. Если [0] - аргументов нет

arg(1) - за что отвечает аргумент 1

\~arg(2) - за что отвечает аргумент 2

...

arg(n) - за что отвечает аргумент n



`snick [id, newName]` - устанавливает никнейм с ID по определенному шаблону ("2" Др. Смирнов)

id - ID игрока, которому будет выдан никнейм

newName - новый никнейм


`respawnanimation(ranim) [wave]` - проигрывает анимацию приезда МОГ/ПХ

wave - вид волны (mtf/ci)


`tesla [0]` - переключает состояние тесла-ворот


`customtaskforce(ctf) [groupName]` - устанавливает название МОГ для админ-гана

groupName - название группы


`facilitycolor(fcolor) [\~hexcolor]` - устанавливает цвет света во всем комплексе

\~hexcolor - hex-код цвета (#eeeeee), если не установить - цвет станет дефолтным


## <a name="admin-gun">Админ-ган</a>:

Упрощает выдачу ролей

ID кастомного предмета - 5 (ci give 5)

Для работы с админ-ганом забиндите соответсвующие пункты в ServerSpecific

Держать в инвентаре могут только администраторы, у остальных он удаляется из инвентаря

Для переключения службы нажмите клавишу, которая у вас забинжена в SCP: SL на выбрасывание предмета

Для переключения подроли нажмите клавишу, которая у вас забинжена в SCP: SL на перезарядку

Чтобы выдать человеку или себе роль нажмите клавишу забинженную вами ранее в ServerSpecific

Чтобы выкинуть админ-ган зайдите в инвентарь и нажмите по нему ПКМ или удалите через админ-панель COM-18
