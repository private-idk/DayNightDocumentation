# Документация для администрации DAYNIGHT HARD RP
## Команды:
cmd(cmdAlias,...,cmdAlias) [arg(1) ~arg(2) ... arg(n)] - функция команды
В [...] находятся аргументы команды, если аргумент помечен тильдой (~), то он не обязателен. Если [0] - аргументов нет
arg(1) - за что отвечает аргумент 1
~arg(2) - за что отвечает аргумент 2
...
arg(n) - за что отвечает аргумент n

snick [id, newName] - устанавливает никнейм с ID по определенному шаблону ("2" Др. Смирнов)
id - ID игрока, которому будет выдан никнейм
newName - новый никнейм

respawnanimation(ranim) [wave] - проигрывает анимацию приезда МОГ/ПХ
wave - вид волны (mtf/ci)

tesla [0] - переключает состояние тесла-ворот

customtaskforce(ctf) [groupName] - устанавливает название МОГ для админ-гана
groupName - название группы
